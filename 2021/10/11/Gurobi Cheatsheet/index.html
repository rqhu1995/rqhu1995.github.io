Gurobi Cheatsheet

  • read the model

    m = read('<the path of the model>')

  • view assignment for each decision variable：

    m.printAttr('X')

  • view all variables

    m.getVars()

  • print the variables(name and solution)

    v[0].varName, v[0].x

  • set lower/upper bound(e.g., set lower bound of v[0] to 10)

    v[0].lb=10

  • 回调函数，参数：model, where

回调函数名可以直接作为参数传给optimize函数
