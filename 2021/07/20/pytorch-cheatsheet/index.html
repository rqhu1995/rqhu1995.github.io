TORCH.UNSQUEEZE

torch.unsqueeze(dim: int)，在dim后加入一个单维度：

tensor = torch.tensor([1, 2, 3, 4])
tensor.shape # torch.Size([4])
tensor_unsqueeze = tensor.unsqueeze(0) # 在第0维度添加一个单维度，值为1
tensor_unsqueeze.shape # torch.Size([1, 4])
tensor_unsqueeze = tensor.unsqueeze(1)
tensor_unsqueeze.shape # torch.Size([4, 1])
