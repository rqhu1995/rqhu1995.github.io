Standard Form Linear Programming

  • n variables

  • 1 objective function

  • m constraints

    $\mathbf{Ax=b}$

  • All varibles are Non-negative

Converting to the standard form

$$
\begin{aligned}
&\text { Maximize } \quad 3 x_{1}-2 x_{2}-4\left|x_{3}\right| \\\
&\begin{aligned}
\text { s. t. } \quad -&x_{1}+2 x_{2} \leq-5 \\\
&3 x_{2}-x_{3} \geq 6 \\\
&2 x_{1} + x_{3} =12 \\\
&x_{1}, x_{2} \geq 0
\end{aligned}
\end{aligned}
$$

Problem 1: Unrestricted free variables

Every real number can be viewed as positive part and negative part.
$$
\begin{aligned}
&x_{i}^{+}= \begin{cases}x_{i}, & \text { if } x_{i} \geq 0 \\\
0, & \text { otherwise }\end{cases} \\\
&x_{i}^{-}= \begin{cases}0, & \text { if } x_{i} \geq 0 \\\
-x_{i}, & \text { otherwise }\end{cases}
\end{aligned}
$$
$x_i=x_i^+ - x_i^-$

either $x^+_i$ is 0 or either $x^-_i$ is 0

$|x_i|=x^+_i+x^-_i$

potential problem: quadratic constraint: $x^+_i\cdot x^-_i=0$

Problem 2: inequality

adding slack variable for $\le$ or minus excess variable for $\ge$

Problem 3: minimization

Minimize the negative

result:

(-)Minimize
$$
-3 x_{1} \quad+2 x_{2} \quad+4 x_{3}^{+}+4 x_{3}^{-}
$$
subject to
$$
\begin{array}{rcccc}
&-x_{1} &+2 x_{2} & & &+x_{4} & &=&-5 \\\
& &\quad3 x_{2}&-x_{3}^{+}&+x_{3}^{-} & &-x_{5} &=&6 \\\
&2 x_{1} & &+x_{3}^{+}&-x_{3}^{-} &&& =&12\\\
&x_{1}, &\quad x_{2}, &\quad x_{3}^{+}, &x_{3}^{-},&x_{4}, &\quad x_{5} &\geq &0
\end{array}
$$

Still problematic：

  • Missing the quadratic constraint
  • Dimensionality increase
  • One orginal solution becomes infinity ($x^*=5=6-1=7-2=$…)
  • Maximize $c|x| (c\ge0)$, unable to solve. (maximizing a convex function is impossible)
